#  AI Influencer Campaign Dashboard

An interactive Streamlit dashboard to manage, filter, and analyze influencer campaigns using data-driven insights.

---

## 📌 Overview
This project helps in simplifying influencer campaign management by allowing users to filter creators, analyze performance metrics, and make structured decisions.

---

## Features

### Dashboard Overview
- Total number of creators
- Average creator score
- Total reach (views)
- Quick data preview

### Campaign Manager
- Create campaigns with:
  - Campaign Code
  - Campaign Name
  - Target Niches
- Filter creators based on niche
- Take actions:
  - Shortlist
  - Reject
  - Backup
  - Pending
- Track campaign decisions
- Export results as CSV

### Insights
- Reach vs Engagement analysis
- Cost vs Creator Score analysis
- Correlation matrix of key metrics

---

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Google Sheets (used as backend data source)

---

## Data Flow
- Data is fetched from **Google Sheets (live backend)**  
- Processed and visualized in the Streamlit dashboard  
- Final campaign results are exported as **CSV files**
